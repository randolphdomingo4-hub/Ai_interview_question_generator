# AI Interview Questions Generator V1

A beginner AI-powered Pyton application that generates interview questions, identify question difficulty, provide sample answers and include what interviewers are looking for based on user input using Google's Gemini API.
## Features
- Accept Target Role
- Accept Experience level
- Generates 5 interview questions
- Rate question difficulty
- Provide Sample Answers
- Include what interviewers are looking for

## Skills Used
- Pyton fundamentals
- Functions
- User input handling
- API integration
- Google Gemini API
- AI application development
## How to run
1. Install requirements

pip install -r requirements.txt

2. Run the application

python app.py

3. Enter:
- Target Role
- Experience Level

4. Review the generated interview questions
